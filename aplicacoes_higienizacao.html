Cruzamento das bases RFB e SEFAZ-MA




O programa

O programa foi desenvolvido para gerar os relatórios solicitados pela equipe da Sefaz-MA, de acordo com os cruzamentos de dados das bases da RFB e da SEFAZ-MA, com formatos de em PDF, XLSX, CSV e HTML.

Link para Download


Gerando os relatórios

  1. Ao abrir o programa, clicando no botão Relatórios, teremos as opções de relatórios


  2. É necessário preparar os dados, clicando no botão Prepara dados RFB x SEFAZ-MA (ICMS) o programa irá pegar os dados das empresas de ambas as bases e deixá-los prontos para a filtragem do relatório.


  3. A filtragem dos dados deve ser feita de ambas as bases, mas caso haja problemas na execução ou atualização no banco, pode-se utilizar somente uma filtragem para agilizar o processo. Ela extrai os dados necessários para o relatório e salva numa tabela do banco de dados.




  4. Os tipos dos relatórios podem ser vistos no botão de informações i




  5. Os botões Somente Sefaz e Somente RFB geram o relatório em tela, com a opção de salvar em vários formatos.




  6. Os botões de exportar em PDF ou XLSX geram o arquivo na pasta onde o programa foi executado.






  7. No caso do relatório da RFB, deve-se escolher a situação cadastral, podendo ser uma ou todas.


    8. *Atenção, devido a grande quantidade de dados para filtrar e extrair, o processo pode demorar até mais de 30 minutos.
